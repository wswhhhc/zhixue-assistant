# 智学助手 — 用户手册

## 一、简介

**智学助手** 是一款 AI 驱动的高等数学学习平台，专注于选择题练习与智能分析。它利用 AI 技术提供批改、错因诊断、相似题生成、学习报告等全链路学习服务。

适用对象：正在学习高等数学（微积分、线性代数等）的学生。

---

## 二、快速开始

### 2.1 启动项目

**方式一：Docker 启动（推荐）**

双击 `docker-start.bat`，首次启动会自动构建镜像，构建完成后访问 `http://localhost:8000` 即可。

**方式二：本地开发启动**

双击 `start.bat`，或分别启动后端和前端：

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000

# 新开终端
cd frontend
npm install
npm run dev
```

## 四、登录与注册

### 4.1 登录

- 输入用户名或邮箱 + 密码
- 填写图片验证码（点击可刷新）
- 勾选"记住我"可保存用户名
- 登录成功后会看到数学符号粒子动画，自动跳转到仪表盘

### 4.2 注册

- 输入用户名、邮箱
- 点击"发送验证码"，到邮箱查收验证码
- 输入验证码和密码（至少 6 位）完成注册
- 注册成功后自动登录

## 五、仪表盘

仪表盘是学习总览页面，包含以下模块：

### 5.1 统计概览

顶部四个统计卡片：
- **今日做题** — 今日完成的题目数量
- **今日正确率** — 今日答题正确百分比
- **总做题数** — 累计完成的题目总数
- **总正确率** — 累计答题正确百分比

### 5.2 打卡与目标

- **打卡** — 每日可打卡一次，记录连续打卡天数，显示本周打卡情况
- **今日学习时长** — 根据做题量估算的学习时间
- **今日目标** — 设置每日做题目标，可点击"修改"调整（范围 1-200 题），进度条显示完成情况

### 5.3 AI 问答

在页面底部的输入框中输入数学问题，AI 会实时流式回答。支持 LaTeX 公式渲染。

- 按 `Enter` 发送，`Shift+Enter` 换行

### 5.4 学习动态

按时间线展示近期的学习活动，包括答题结果和打卡记录。

### 5.5 学习趋势

近 14 天正确率变化趋势折线图。

### 5.6 知识点掌握度

雷达图展示各知识点的掌握百分比。

### 5.7 最近错题

最近答错的题目列表，点击可查看详情。

---

## 六、刷题

刷题页面提供四种答题模式：

### 6.1 模式选择

| 模式 | 说明 |
|------|------|
| **顺序答题** | 按题目 ID 顺序浏览，支持上一题/下一题，可"从头开始" |
| **随机答题** | 从题库中随机抽题，点击"换一题"跳过 |
| **智能推荐** | AI 根据你的薄弱知识点推荐题目，显示推荐原因和掌握度 |
| **知识点练习** | 选择特定知识点，集中练习该知识点的所有题目 |

### 6.2 答题流程

1. 题目以 LaTeX 格式展示，选项为 A/B/C/D 单选
2. 选择一个选项后点击"提交答案"
3. AI 实时分析答案并返回批改结果
4. 自动跳转到结果页面查看详细分析

### 6.3 收藏题目

点击题目右上角的星标图标可收藏/取消收藏题目。

### 6.4 重做错题

在错题本页面点击"重做所有错题"，可在刷题页面按顺序重做历史错题。

---

## 七、答题结果

提交答案后进入结果页面，包含以下内容：

### 7.1 结果总览

- 显示"✓ 回答正确"或"✗ 回答错误"
- 错误类型标签（如"概念理解偏差""计算错误"等）

### 7.2 题目回顾

展示原题内容、你的答案（红色标记）、正确答案（绿色标记）。

### 7.3 AI 错因分析

AI 对错误原因进行详细分析。

### 7.4 解题步骤

AI 提供完整的解题过程。

### 7.5 学习建议

针对薄弱点给出学习方向和建议。

### 7.6 巩固练习

AI 自动生成一道相似题目供巩固练习。

### 7.7 AI 生成相似题

点击"AI 生成相似题"按钮，AI 会基于本题生成一道同知识点的相似题并跳转练习。

---

## 八、题库

题库页面以表格形式展示所有题目，支持：

- **按知识点筛选** — 下拉选择知识点过滤题目
- **查看题目** — 点击任一行可直接跳转到刷题页面练习该题
- **编辑题目** — 点击编辑图标修改题目内容、选项、答案、知识点
- **删除题目** — 点击删除图标移除题目

---

## 九、上传题目

支持两种方式添加新题目到题库：

### 9.1 拍照上传

1. 切换到"拍照上传"标签
2. 点击或拖拽图片到上传区域
3. AI 自动识别图片中的题目内容、选项、答案
4. 识别后可编辑修正
5. 点击"确认入库"提交，AI 会自动审查完整性
6. 如审查未通过，可手动修改或点击"AI 自动修改"

### 9.2 手动输入

1. 切换到"手动输入"标签
2. 填写题目内容（支持 LaTeX 公式，用 `$...$` 包裹）
3. 填写四个选项、正确答案、知识点
4. 点击"提交"后进入编辑确认页面

### 9.3 预览

填写题目后，页面底部会实时预览 LaTeX 渲染效果。

---

## 十、错题本

错题本自动收录所有答错的题目。

### 10.1 错题列表

- 按时间倒序排列，支持分页
- 每条记录显示知识点、题目内容摘要和答题时间
- 点击任一条目进入详情

### 10.2 错题详情

- **原题** — 展示题目和选项，标记你的答案（红色）和正确答案（绿色）
- **AI 错因分析** — 错误原因分类和详细分析
- **解题步骤** — 完整的解题过程
- **学习建议** — 针对性学习指导
- **巩固练习** — AI 生成的相似巩固题

### 10.3 错题操作

- **重做此题** — 跳转到刷题页面重新练习
- **删除记录** — 从错题本中移除记录
- **返回错题本** — 回到列表

### 10.4 重做所有错题

在错题本列表页点击"重做所有错题"，进入刷题模式逐题重做。

---

## 十一、学习报告

学习报告生成全面的学习数据分析，支持打印和导出。

### 11.1 报告内容

- **统计概览** — 总做题数、总正确率、总错误数、今日正确率
- **知识点掌握度** — 雷达图可视化各知识掌握情况
- **正确率趋势** — 近 14 天正确率变化折线图
- **错误类型分布** — 饼图展示各类错因占比（概念偏差、计算错误、粗心失误等）
- **薄弱知识点** — 掌握度最低的知识点列表
- **AI 学习建议** — AI 根据学习数据生成个性化建议

### 11.2 导出功能

- **打印/导出 PDF** — 使用浏览器打印功能保存为 PDF
- **导出数据** — 下载 JSON 格式的学习数据

## 十四、管理员后台

### 14.1 管理员登录

访问 `/admin/login` 进入管理员登录页面，使用管理员账号登录。

> 管理员账号需要在数据库中手动设置用户的 `role = 'admin'`。

### 14.2 系统概览

管理员仪表盘展示：

- **核心指标** — 用户总数、题目总数、答题总数、今日答题数、会员用户数、今日活跃用户
- **待处理事项** — 待审核题目数、待支付订单数、生效兑换码数、24 小时新增用户
- **用户行为趋势** — 近 7 天答题量、活跃用户、用户上传量的变化折线图
- **快捷操作区** — 快速跳转到内容审核、支付订单、兑换码管理、用户管理、题库管理、错题分析
- **做题趋势图** — 近 7 天每日答题量
- **知识点正确率** — 各知识点正确率横向柱状图
- **题目来源分布** — 系统 / 用户上传 / AI 生成的占比
- **用户会员比例** — 免费用户 vs 会员用户占比

### 14.3 用户管理

- 展示所有普通用户列表（排除管理员）
- 支持按用户名/邮箱搜索
- 可编辑用户的会员状态和有效期
- 点击可查看用户的学习详情（统计数据、知识点掌握度、高频错题、错误类型分布、最近做题记录）

### 14.4 题库管理

- 展示全部题目（系统预置 + 用户上传 + AI 生成）
- 支持按来源、知识点、关键词搜索筛选
- 可编辑题目内容、选项、答案、知识点
- 可删除题目（同时清理关联的答题记录和收藏记录）

### 14.5 内容审核

- 用户上传的题目需要管理员审核后才能进入学生端题库
- 支持逐题审核（通过/驳回）和批量审核
- 审核结果会自动通知题目上传者

### 14.6 兑换码管理

- 查看所有会员兑换码
- 批量生成兑换码（设置有效期和最大使用次数）
- 启用/禁用兑换码

### 14.7 支付管理

- 查看所有支付订单
- 按状态筛选（待支付 / 已完成 / 已关闭）
- 查看订单详情

### 14.8 统计分析

- **高频错题排行榜** — 按错误次数排序，查看各题目的错误率
- **错误类型分布** — 全平台错误类型的饼图分布
- **薄弱知识点排名** — 按正确率升序排列的知识点列表

## 十五、常见问题

### 15.1 无法登录？

- 检查后端是否已启动（后端运行在 `http://localhost:8000`）
- 验证码图片是否正常加载（可点击刷新）
- 测试账号：wsw / 123456

### 15.2 AI 功能没有响应？

- 检查 `backend/.env` 中的 LLM API 配置是否正确
- 确保有可用的网络连接
- API Key 余额是否充足

### 15.3 公式显示异常？

- 项目使用 KaTeX 渲染 LaTeX 公式
- LaTeX 内容需要用 `$...$` 包裹
- 检查公式语法是否正确

### 15.4 图片上传不成功？

- 支持 PNG、JPG 格式
- 确保图片中的文字清晰可辨
- 图片不要过大
